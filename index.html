1. Working with Math Object


When you click this button, JavaScript will generate a random number and display the number in the box. Each time you click the button, JavaScript will displays a different random number.

Here is the code that generates and displays the random number.


    <html>
        <button onclick="randomNumberMaker()">
            Display a random number</button>
        <span id="demo-1" class="numbers"></span>
    </html>

    <script>
        function randomNumberMaker() {
            var randomNumber = Math.random();
            document.getElementById("demo-1").innerHTML = randomNumber;
        }
    </script>

Math.random

JavaScript includes a Math object that has a number of methods.
You can access any of the Math object's methods following this syntax: Math.methodName();

The Math.random() method will generate a random decimal number between 0 and 1.

  var randomNumber = Math.random();

Create or declare a variable named randomNumber and assigned to it will be the random number generated by Math.random(). But, what if you want to generate whole random numbers, say between 0 and 10?

Math.floor

We can use another one of the Math object's methods to get rid of the numbers to the right of the decimal: Math.floor(). Math.floor() rounds a number down to the nearest whole number.


    <html>
        <button onclick="randomNumberMaker2()">
            Display a random whole number between 0 and 10</button>
        <span id="demo-2" class="numbers"></span>
    </html>

    <script>
        function randomNumberMaker2() {
            var randomNumber = Math.floor(Math.random() * 11);
            document.getElementById("demo-2").innerHTML = randomNumber;
            }
    </script>

This code also shows you can nest one Math method inside of another! Here, JavaScript will first generate a random number between 0 and 1, then multiply it by 11, then submit that value to the Math.floor() function. When you use this method, by the way, it is possible to generate the same random number more than once in a row.
